Налаштування проекту

Цей документ описує підключення та використання TailwindCSS, налаштування оптимізації зображень та стилів.

Налаштування:

Налаштування збірки здійснються в файлі template.config.js в корені проекту.

// template.config.js

export default {
   // Вмикнути чи вимкнути TailwindCSS
   isTailwind: true,
   // Чи використовується PHPMailer, чи підіймати PHP сервер
   isPHPMailer: false,
   // Чи показувати навігаію між існуючими сторінками (в dev режимі)
   isDevNavigation: true,

                  
   imgQuality {

                  
      ...Налаштування оптимізації JPEG, PNG, WEBP

                  
   }

                  
   HTMLVariables {

                  
      ...Глобальні HTML змінні

                  
   }

                  
   PHPserver {

                  
      ...Налаштування PHP сервера

                  
   }

                  
   aliases {

                  
      ...Налаштування псевдонімів (аліасів)

                  
   }

                  
   // Вмикнути чи вимкнути відносні псевдоніми

                  
   aliasesIsRelative: false,

                  
}
