Структура проекту

Цей документ описує основну структуру проекту. 

|--| dist/ ------------------------ Зібраний проект
|--| src/ ------------------------- Папка з похідними файлами
|-----| data/ --------------------- Всі дані в json
|--------| products.json ---------- json файл з даними
|-----| html/ --------------------- Файли HTML
|--------| components/ ------------ Компоненти
|-----------| Header/
|--------------| Header.html
|--------------| header.scss
|-----------| Footer/
|--------------| Footer.html
|--------------| footer.scss
|--------| ui/ -------------------- UI елементи (кнопки, інпути...)
|-----------| Button/
|--------------| Button.html
|--------------| button.scss
|-----| js/ ----------------------- Скрипти проекту
|--------| main.js ---------------- Головний файл скриптів
|--------| custom.js -------------- Кастомні скрипти
|-----| css/ ---------------------- CSS Стилі TailwindCSS
|--------| main.css --------------- Основний файл стилів TailwindCSS
|-----| scss/ --------------------- SCSS Стилі
|--------| base/ 
|-----------| base.scss ----------- Загальний файл підключення базових стилів
|-----------| layout-grid.scss ---- Адаптивна grid сітка
|-----------| mixins.scss --------- Файл з міксинами
|-----------| reset.scss ---------- Базове обнулення стилів
|-----------| variables.scss ------ CSS змінні
|--------| fonts/
|-----------| fonts.scss ---------- Шрифти
|-----------| icons.scss ---------- Іконки іконочного шрифта
|--------| libs/ ------------------ Стилі бібліотек (swiper, lightgallery...)
|--------| settings/
|-----------| media.scss ---------- Розміри брейкпоінтів
|-----------| containers.scss ----- Налаштування обмежуючих контейнерів
|-----------| params.scss --------- Глобальні налаштування проекту
|--------| inc.scss --------------- Імпорт міксинів, параметрів, змінних у main.scss
|--------| global.scss ------------ Глобальні стилі
|--------| main.scss -------------- Головний файл проекту
|-----| assets/ ------------------- Файли проекту
|--------| files/ ----------------- Копіюються при білді
|--------| fonts/ ----------------- Шрифти Woff2
|--------| img/ ------------------- Зображення
|--------| video/ ----------------- Відео
